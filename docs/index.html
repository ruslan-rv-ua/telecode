telecode 0.2.0

Програма для демонстрації коду.

Встановлення і запуск.

  1. Завантажте zip-архів з програмою.
  2. Розпакуйте завантажений zip-архів в обрану вами або рекомендовану викладачем теку.
  3. Перейдіть у теку з розпакованою програмою telecode-0.2.0.
  4. Запустіть файл telecode.exe.

Відкрити у редакторі

Файл відкривається у текстовому редакторі Notepad++.

Щоб у редакторі поновити код, натисніть Ctrl + F5.

Щоб запустити код на виконання, натисніть Alt + Shift + F5
